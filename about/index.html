Hi!

This is Vito Chin and this is my blog about the making and using of software.

Marc Andreessen explained Why Software is Eating the World many years ago.

It is, it still is, and I think it is mostly a good thing.

This site is about the software bites I take, while it eats the world :)

It’s not as cheesy as it sounds.

Expect to read about my journey in software from 2020 onwards.

For past texts (1991-2019) and more structured contents about me, have a look at my legacy site.

This site will mostly document my software journey as an App Innovation Cloud Solution Architect at Microsoft. The gist of what I do: helping partners and customers build great software by (re)using Azure services.
